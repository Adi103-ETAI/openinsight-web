<h1 align="center">OpenInsight</h1>

<p align="center">
  <strong>AI-Powered Clinical Decision Support for Indian Doctors</strong>
</p>

<p align="center">
  A multi-page marketing website built with Next.js, React, TypeScript, and Tailwind CSS.
</p>

## Pages

- `/` — Home
- `/product` — Product Overview
- `/for-doctors` — For Doctors
- `/evidence` — Clinical Evidence
- `/about` — About Us
- `/early-access` — Early Access Signup

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **UI Library:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
